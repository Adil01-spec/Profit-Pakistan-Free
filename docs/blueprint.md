# **App Name**: Profit Pakistan Pro

## Core Features:

- User Authentication: Secure user login and signup using Firebase Authentication with Google Sign-In, storing user profiles in Firestore.
- Product Launch Planner: Allows users to input product metrics and calculate profit margin, breakeven ROAS, and cost per sale, providing color-coded profitability feedback.
- Ad Feasibility Calculator: Computes total costs (Shopify, transaction, bank fees, ad spend) and net profit/loss, suggesting breakeven points with visual indicators based on user input and bank fee approximations.
- Data Persistence: Store all user data including Shopify plans, product analyses and ad profitability assessments inside of a Firestore database.
- Data Visualization: Generate visualizations, metrics and suggestions for improving profitability of Pakistan-based e-commerce stores.
- Report History: Provide a history screen that will list the history, name and key details of past reports.
